# Frontend UI Patterns

Use common, predictable controls:

- Segmented controls for device and export mode.
- Tabs for inspector categories.
- Buttons for direct actions.
- Selects for semantic option sets.
- Text inputs and textareas for copy.

Avoid professional design-tool language in beginner workflows.

## Adjacent Product Lessons

Current EasyFrontEnd UI implications:

- Open into the visual editor by default.
- Keep AI generation and optimization as auxiliary entry points.
- Prefer section-level drag, point, choose, type, and preview over prompt-first editing.
- Keep responsive controls semantic.
- Keep export generated from the same DSL as the canvas.
