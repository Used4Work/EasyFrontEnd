# Quality Score System

The quality score gives beginner users a clear sense of whether a landing page is ready to preview or export.

## MVP Approach

The first version uses deterministic rules against the DSL. This keeps feedback explainable and testable.

## Scoring Dimensions

- Structure Completeness
- Visual Hierarchy
- CTA Clarity
- Responsive Readiness
- Accessibility Basics
- Content Clarity
- Export Readiness

Suggestions should identify the target section when possible and use plain language rather than design jargon.
