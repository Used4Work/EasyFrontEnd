# Quality Score

The MVP quality score is deterministic and DSL-based. It is meant to teach users what makes a landing page clearer and more shippable.

## Dimensions

- Structure Completeness
- Visual Hierarchy
- CTA Clarity
- Responsive Readiness
- Accessibility Basics
- Content Clarity
- Export Readiness

The score should be explainable. Every deduction should map to a concrete issue or suggestion.

## Current Rules

The implemented MVP checks required landing-page modules, Hero CTA strength, final CTA presence, mobile density risks, theme contrast, placeholder-like content, and DSL validation readiness.
