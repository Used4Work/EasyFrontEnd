# Product Sense

EasyFrontEnd is not a prompt-to-page toy. The product value is the loop:

AI draft -> visual confirmation -> semantic editing -> quality feedback -> reliable export.

The user should feel that AI helps them start and improve, while they remain in control of the final page.

## Core Audience

Beginner founders, educators, consultants, creators, and operators who need a credible landing page but do not know frontend design or code.

## Product Bet

Controlled freedom beats unlimited freeform editing for this audience. A smaller set of good, semantic choices creates better outcomes than exposing every CSS capability.
